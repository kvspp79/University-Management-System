# University Management System

This project is developed using Java Swing and Eclipse WindowBuilder.

## Modules
- Administrative Login
- Staff Login
- Student Login

## Main Features
- Student registration and record saving
- Separate Admin, Staff, and Student portals
- Attendance management
- Exam paper creation
- Student exam submission
- Marks publishing
- Complaint management
- Payment status management
- Student analytics and graphs

## Technologies Used
- Java
- Swing
- Eclipse WindowBuilder
- File Handling using TXT files

## Project Logic
- Admin creates and manages student records
- Staff manages attendance, exams, complaints, and marks
- Students can view their own profile, attendance, marks, timetable, and write exams

## Future Scope
- Web-based deployment
- QR-based access
- Database integration
- Mobile-friendly portal
